Le programme

De spectateur
à opérateur.

5 modules. Des cas d'usage réels. Vous repartez avec des outils qui tournent — pas des slides.

MODULE 1 Comprendre le terrain
45 min +

L'écosystème IA en 2025 : ce qui compte, ce qui est du bruit. Pas un glossaire — une carte mentale actionnable.

  • Les modèles et leurs différences : Claude, GPT-4o/o3, Gemini 2.5, DeepSeek, Mistral — qui fait quoi, pour quel usage, à quel prix
  • Pourquoi le marché du logiciel est en train de basculer : l'effondrement du coût marginal du code
  • Les concepts clés introduits au fil de l'eau : LLM, tokens, context window, RAG, fine-tuning, agents, hallucinations
  • Les sources pour rester à jour : newsletters, comptes à suivre, comment filtrer le signal du bruit
Exercice : chaque participant identifie 3 tâches répétitives de sa semaine type. Cette liste sera le fil rouge du bootcamp.
MODULE 2 Maîtriser la conversation avec l'IA
1h30 +

Savoir parler à un LLM, c'est la compétence socle. Tout le reste en dépend.

  • Prompting : anatomie d'un bon prompt, prompt chaining, system prompts, Custom GPTs, chain-of-thought, meta-prompting
  • Fonctionnalités puissantes : Deep Research, analyse de fichiers, Canvas/Artifacts, Claude Code en mode tasks
  • Outils satellites : Granola, Fyxer, Perplexity, NotebookLM, Jasper — et comment les intégrer dans votre quotidien
Exercice : résoudre en live une de vos 3 tâches avec Claude, en itérant sur le prompt jusqu'au résultat.
MODULE 3 Automatiser vos workflows
2h +

Connecter vos outils entre eux pour que les tâches se fassent sans vous.

  • Make / n8n / Zapier : trigger → action → filtre → branchement. On construit ensemble : tri d'emails, synthèse de réunion → Notion, reporting hebdo automatique
  • Claude Cowork / Computer Use : générer un deck PowerPoint, nettoyer un Excel, préparer 50 emails de prospection personnalisés
  • Agents et MCP : donner à Claude un accès direct à votre agenda, Drive, Slack, Notion
Exercice : chaque participant met en place une automatisation fonctionnelle sur un de ses cas réels.
MODULE 4 Construire vos propres outils
2h +

Vous décrivez ce que vous voulez en français. L'IA écrit le code. Vous avez un outil qui tourne.

  • Le vibe coding : Lovable, Bolt.new, v0 (Vercel), Cursor, Windsurf, Claude Code — chaque plateforme par niveau de complexité
  • Atelier construction : dashboard de prospection, formulaire intelligent → Airtable, outil d'analyse de documents, landing page
  • Déploiement : Vercel, Netlify, Lovable hosting — mettre en ligne et partager avec son équipe
Exercice : chaque participant définit un outil manquant dans son quotidien et en lance la construction.
MODULE 5 Déployer dans votre organisation
45 min +

Repartir avec un plan d'action, pas juste de l'enthousiasme.

  • Identifier les quick wins (déployables cette semaine) vs. les projets structurants (1–3 mois)
  • Convaincre en interne : démonstrations, ROI chiffré, adoption progressive par l'exemple
  • Risques et garde-fous : hallucinations, confidentialité, RGPD, dépendance fournisseur
  • Gouvernance minimale : charte d'usage IA, bonnes pratiques équipe, veille sur les évolutions
  • Construire sa stack personnelle : quels outils garder, comment rester à jour sans y passer 3h/jour
